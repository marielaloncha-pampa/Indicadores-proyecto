## Performance & Financial Tracking
In addition to the simulator, the tool monitors project execution:
* **Incurred vs. Budgeted:** Real-time tracking of actual spending against the simulated budget.
* **Progress KPIs:** Visual representation of work completion status and variance analysis.
* **Resource Allocation:** Detailed breakdown of costs across different project stages.

## Technical Skills Demonstrated
* **Advanced DAX:** Complex measures used to power the dynamic simulator logic.
* **User-Centric Design:** Focused on interactivity and "what-if" analysis.
* **Data Modeling:** Efficient structure to handle dynamic calculations.
